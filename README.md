# Stock Price Prediction Using Recurrent Neural Networks (RNN)

## Overview
This project predicts future stock prices using Recurrent Neural Networks (RNN) and historical stock market data.

## Features
- Stock price forecasting
- Data preprocessing and normalization
- RNN-based prediction model
- Future price prediction
- Interactive Streamlit dashboard
- Performance evaluation metrics

## Tech Stack
- Python
- TensorFlow
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- Plotly

## Project Workflow
1. Upload stock dataset
2. Data cleaning and preprocessing
3. Sequence generation
4. RNN model training
5. Prediction and evaluation
6. Future stock forecasting

## Results
The model learns historical price patterns and predicts future stock prices with visualization support.
